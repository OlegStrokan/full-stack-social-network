# Full stack social network

- social-network - main backend part
- chat-service - microservice for main app
- frontend - frontend part

## Backend stack:

* Nest.js
* PostgreSQL
* Sequelize
* Typeorm
* Swagger
* Docker
* Web sockets/socket.io
* RabbitMQ

## Frontend stack:

* React
* Redux-toolkit
* Redux-saga
* Material UI
* Socket.io
* React testing library
* Cypress
