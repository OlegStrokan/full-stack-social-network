### V_4.1.0

## Frontend stack:
* React - Main library
* Redux-toolkit - State manager
* Material UI - UI library
* Socket.io - Websockets library
* React testing library - Unit testing
* Cypress - E2E testing


## Running the app

```bash
# install 
$ yarn install
```

```bash
# development
$ yarn start
```

## Testing the app
```bash
# unit testing
$ yarn test
```

```bash
# cypress testing
$ yarn cypress
```

```bash
# cypress testing with running app
$ yarn test:cypress
```
