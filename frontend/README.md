### V_4.8.0

## Frontend stack:
* React - *main library*
* Redux-toolkit - *state manager*
* Material UI - *UI library*
* Socket.io - *websockets library*
* React testing library - *unit testing*
* Cypress - *e2e testing*


## Running the app

```bash
# install 
$ yarn install
```

```bash
# development
$ yarn start
```

## Testing the app
```bash
# unit testing
$ yarn test
```

```bash
# cypress testing
$ yarn cypress
```

```bash
# cypress testing with running app
$ yarn test:cypress
```
